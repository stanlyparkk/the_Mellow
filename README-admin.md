# 관리자 사용 가이드

이 문서는 Sanity Studio에서 더멜로우 사이트 콘텐츠를 수정하는 방법을 간단히 설명합니다.

## 처음 만들 문서

- `Site Settings` 1개
- `Gallery Main Category` 여러 개
- `Gallery Style Category` 여러 개
- `Gallery / Works` 여러 개
- `Menu / Price` 여러 개
- `Reviews / Clips / Blog Links` 여러 개
- `News / Events` 여러 개

## Site Settings

- 브랜드명
- 메인 배너 제목 / 설명
- 100% 예약제 안내 문구
- 주소 / 영업시간 / 주차 / 오시는 길
- 네이버 지도 / 네이버톡톡 / 카카오톡 링크
- 주문서 양식 / 가격표 링크
- About 소개글
- Gallery 페이지 제목 / 설명
- Menu 페이지 제목 / 설명
- Order 진행 안내 단계
- 연락처 / 이메일 / 인스타그램 링크

## Gallery / Works

- 작업물 제목
- 목록용 짧은 설명
- 콘텐츠 유형(사진 / 릴스)
- 상세 설명
- 등록일 또는 시즌 날짜
- 주문 메모 / 행사 메모
- 대표 이미지와 갤러리 이미지
- 메인 카테고리와 스타일 카테고리
- 메인 노출 여부

## Menu / Price

- 메뉴명
- 가격
- 설명
- 구성 또는 옵션
- 추가 안내 문구
- 뱃지 문구
- 정렬 순서

## Reviews / Clips / Blog Links

- 리뷰, 클립, 블로그, 공지 링크를 카드 형태로 등록합니다.
- 외부 리뷰 사진이나 블로그 본문은 복사하지 말고 링크와 짧은 설명 중심으로 등록하는 것을 권장합니다.

## News / Events

- 시즌 주문 안내
- 이벤트 안내
- 원데이 클래스 일정 안내
- 외부 주문서나 공지 링크 연결
